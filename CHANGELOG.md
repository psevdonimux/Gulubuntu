# Сочетания клавиш

1. **alt + z** - Включение/Отключение интерфейса
2. **shift + f1** - Включение/Отключение фильтра синего
3. **shift + f2** - Включение/Отключение монохрома
4. **print screen** - Скриншот экрана по пути `/home/gulubuntu/Изображения`
5. **pause break** - Включить/Отключить съёмку видео с экрана по пути `/home/gulubuntu/Видео`

# Команды

1. `webapp ссылка` - Открыть сайт в окне без лишнего интерфейса
2. `imgf` - Сжать фото в формат webp по пути `/home/gulubuntu/Изображения`
3. `vf` - Сжать видео в формат webm по пути `/home/gulubuntu/Видео`
4. `bluefilter` - Включить/Отключить фильтр синего
5. `okr` - Включение/Отключение интерфейса
6. `screenshot` - Скриншот экрана
7. `screenrecorder` - Включить/Отключить съёмку видео с экрана
8. `tpk` - Таймер для программ
9. `monochrome` - Включить/Отключить монохром
10. `saturation` - Установить насыщенность монитора(-ов)
11. **Режимы питания:**
	- `powersave` - Максимальная экономия
	- `balance` - Баланс между производительностью и потреблением (Больше в сторону экономии)
	- `medium` - Средний больше ориентирован на производительность (Малая экономия энергии)
	- `performance` - Максимальная производительность (Нет экономии энергии)
12. `settings` - Интерфейс с настройками
