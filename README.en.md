# Gulubuntu
Gulubuntu is a distribution that, being based on Lubuntu, strives for minimalism and simplicity while maintaining ease of use.

# Goals
1. Simplifying usage for beginners.
 - Create an intuitive graphical interface.
 - Minimize the need to work with command line.
 - Implement graphical shell for many system settings.

2. Resource consumption optimization.
 - Minimize system RAM consumption.
 - Reduce CPU load in idle mode.
 - Optimize disk space by removing unused components.
 - Improve energy efficiency for longer battery life.

3. Expanding "out of the box" functionality
 - Pre-install a basic set of packages based on your interests.
 - Provide lightweight yet necessary settings in one place.
 - Create automatic scripts for everyday tasks.

# Contributing
Gulubuntu is an open source project and we welcome everyone's participation! As Lubuntu's slogan said: it doesn't matter if you're 13 or 82. Regardless of your age and experience level, you can contribute to the distribution's development.

# How can you help?

 - Development — improving code, fixing bugs, adding new features.

 - Testing — finding errors, testing on various hardware.

 - Documentation — writing and improving documentation, creating guides.

 - Design — working on interface, icons, themes.

 - Translations — localization to other languages.

 - Community — helping new users, community moderation.

 - Feedback — improvement suggestions, bug reports.
